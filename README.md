# Orexin Receptor (OX2R) Modulation using Molecular Docking and MOOSE

## Overview

This project integrates:

- Molecular docking
- Docking validation
- Protein–ligand interaction analysis
- Hodgkin–Huxley neuronal simulation in MOOSE

to explore qualitative effects of OX2R modulation on neuronal excitability.

## Workflow

Docking
↓
Interaction Analysis
↓
Representative Ligands
↓
Qualitative Modulation Scenarios
↓
MOOSE Simulation
↓
Spike Analysis

## Results

- Hesperidin showed stronger predicted binding than Apigenin.
- Lemborexant was used as a reference antagonist.
- Reduced excitatory drive decreased neuronal firing in the MOOSE model.

## Software

- PyMOL
- AutoDock Vina
- Discovery Studio Visualizer
- MOOSE
- Python

NOTE: The MOOSE simulations represent qualitative scenarios of altered orexin-mediated excitatory drive. Simulation parameters were selected to explore plausible downstream neuronal responses and are not directly derived from molecular docking scores.
